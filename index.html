Controlos
Mouse: rodar / zoom / mover camara
W / S: frente / tras
A / D: esquerda / direita
Q / E: virar esquerda / direita
Espaco / Shift: subir / descer
T: ligar e expandir / desligar e retrair bracos
1 / 2 / 3: topo / lateral / frontal
4 / 5 / 6: fixa ortografica / fixa perspectiva / movel
H: mostrar / ocultar helpers
Estado: ligado
